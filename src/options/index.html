The Preferred Instance option must be configured before this add-on will work. If you don't have a preference, press the Use Recommended Instance button to confirm you're happy to use the official instance hosted at libravatar.org.

Whenever you create or update a contact, this add-on will make an outgoing network request to your preferred instance to check for an avatar.
